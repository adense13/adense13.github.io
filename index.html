What is MultiSnake?

MultiSnake is a multiplayer game based on the popular game Snake. Two or more players will control their own snake with the goal of reaching and eating apples in order to grow. All while avoiding colliding with the edges of the game level.. and other players!

Instructions

1. Setup

  • 1.1 Download the program

    • Download here

  • 1.2 Connecting to existing server

    • Run View.java in the ClientSide-package or run MultiSnake.jar

  • 1.3 Host your own server

    • Go to Level.java. Change address-variable to localhost if you run server and client on the same machine. Otherwise use the IP-address of the computer your server is on.
    • Run ServerMain.java (on the server machine) and keep it on
    • Run View.java in the ClientSide-package or run MultiSnake.jar

2. How To Play

  • 2.1 Controls

    • UP-ARROW - the snake will move up
    • DOWN-ARROW - the snake will move down
    • LEFTT-ARROW - the snake will move to the left
    • RIGHT-ARROW - the snake will move to the right

  • 2.2 Game Rules

    • Collect red apples to grow.
    • Avoid collision with YOUR OWN and OTHER PLAYERS' snakes.
    • You cannot turn the snake backwards (relative to the direction of its head)

Development

We used Java's Swing to build a GUI. The Client only listens to input from the keyboard and the server and renders graphics. The Server receives input from clients and makes all computations and handle the game model. Communication between the server and client was handled using TCP.
