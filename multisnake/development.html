Purpose

MultiSnake is our network programming project for the course EDA095. A combination of game mechanics design and network communication between clients and a server felt like an interesting challenge that would produce a fun end result.

Development Process

GitHub enabled us to collaborate and work on the project from multiple computers. This way, we could assign certain modules of the project to specific members. Though we often met up and coded together on the same computer.

Structure

We used Java's Swing to build a GUI. The Client only listens to input from the keyboard and the server and renders graphics. The Server receives input from clients and makes all computations and handle the game model. Communication between the server and client was handled using TCP.

Evaluation

The project was interesting and allowed for a great deal of network programming to be learned. Most of the project work flowed along smoothely. The most difficult part was to get the transport protocols right.
